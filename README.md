# Simple Banking System (Java)

A console-based banking application built using Java that simulates basic banking operations through a menu-driven interface.

## Features

- PIN Authentication with 3 login attempts
- Check Account Balance
- Deposit Money
- Withdraw Money
- Transaction History
- Mini Statement
- Deposit and Withdrawal Counters
- Last Transaction Tracking
- Account Lock after 3 incorrect PIN attempts
- Menu-driven User Interface
